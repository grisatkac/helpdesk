This is the start page of the introduction to my project

Help desk can help employees of Organisation communicate and make tasks. The project is under development stage.

Let's talk it detail about project. This project consist of: users, departments, tickets, orders. I want to improve project using databases, JWT tokens in future. Jsp pages will be added when backend code has been done.

Components of help desk

Tickets
Ticket needs for describing task. User can create, complete, update, delete, redirect to another user ticket.
User
For using help desk you must be user of this system. User can have some roles: user, admin. User is the part of department.
Queues
Queue have one logical type of tickets. Queues consists of: name of queue, employees who work with this type of tickets. For example in one department 5 employees can be in A queue and another 5 employees can be in B queue.
Department
Department consist of pepole with the same scope of tasks. For example proggers, system administrators, economists.
